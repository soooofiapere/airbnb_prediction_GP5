# Airbnb Prediction GP5

Проект посвящён прогнозированию уровня коммерческой эффективности объявлений Airbnb в Амстердаме.

## Цель проекта

Цель работы — построить модель, которая по данным объявления и отзывам гостей помогает определить, будет ли объект Airbnb коммерчески успешным относительно своей цены

В проекте используются два типа данных:

* табличные признаки объявлений;
* текстовые данные из отзывов.

Источник данных: [Inside Airbnb](https://insideairbnb.com/get-the-data/)

## Структура репозитория

```text
airbnb_prediction_GP5/

│   └── airbnb_train.ipynb

└── README.md
```

## Описание ноутбуков

### `notebooks/airbnb_eda.ipynb`

В этом ноутбуке проводится первичный анализ данных и подготовка датасетов.
Основные шаги:

* загрузка данных Airbnb;
* анализ пропусков и типов признаков;
* обработка табличных данных;
* подготовка текстов отзывов;
* формирование целевой переменной;
* сохранение подготовленных файлов для обучения моделей.

### `notebooks/airbnb_train.ipynb`

В этом ноутбуке проводится обучение и сравнение моделей.
Используются два направления:

* модели для табличных данных;
* модели для текстовых данных.

Эксперименты логируются с помощью MLflow.

## Используемые модели

Для табличных данных были обучены несколько вариантов MLP:

* простая MLP;
* более глубокая MLP;
* MLP с BatchNorm и Dropout;
* MLP с L2-регуляризацией и Dropout.

Для текстовых данных были протестированы:

* Embedding + GlobalAveragePooling;
* Text CNN;
* GRU.


## Как запустить проект

1. Скачать данные с сайта [Inside Airbnb](https://insideairbnb.com/get-the-data/).
2. Поместить данные в папку `data/`.
3. Запустить ноутбук `notebooks/airbnb_eda.ipynb`.
4. После подготовки данных запустить `notebooks/airbnb_train.ipynb`.
5. При необходимости посмотреть логи экспериментов в папке `mlruns/`
